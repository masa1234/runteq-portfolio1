## GitHub Projects 登録手順

1. リポジトリの「Projects」タブ → 「New project」
2. 「Board」ビューを選択
3. カラム構成：`Todo` / `In Progress` / `Done`
4. 各ISSUEを作成し、ラベル・SP（マイルストーンまたはカスタムフィールド）を設定
5. Phase順（0→1→2→3→4）に着手する
